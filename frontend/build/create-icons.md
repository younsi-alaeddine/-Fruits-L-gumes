# Instructions pour créer les icônes PWA

Les icônes suivantes sont requises :
- icon-192.png (192x192 pixels)
- icon-512.png (512x512 pixels)

## Options pour créer les icônes :

1. **Outils en ligne** :
   - https://realfavicongenerator.net/
   - https://favicon.io/
   - https://www.pwabuilder.com/imageGenerator

2. **Créer manuellement** :
   - Utilisez un éditeur d'image (Photoshop, GIMP, etc.)
   - Créez une icône verte (#28a745) avec le texte "F&L" ou votre logo
   - Exportez aux tailles 192x192 et 512x512

3. **Icônes temporaires** :
   - Pour tester rapidement, utilisez n'importe quelle image PNG
   - Redimensionnez-la aux tailles requises
